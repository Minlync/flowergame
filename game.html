Creazy Flowers

I used the for loop function and the random function to create these crazy flowers. In the flower.js page, I set two functions: one is 'move' and the other is 'show'. So, when the user presses the spacebar and hits the flower, it will trigger three functions: 'reduce', 'key press', and 'sound.play'. When the user hits the flower, the shaking flower's size will reduce based on the frequency of the user. Meanwhile, the console will log "HIT" and indicate user hit acurate position of flower element.
